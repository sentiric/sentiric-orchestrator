# 🧠 AI SYSTEM INSTRUCTIONS FOR ORCHESTRATOR
1. **Teknoloji:** Rust 1.84+, Axum, Bollard (Docker API).
2. **İntihar Yasağı:** Orchestrator (Nexus Governor) kendini ASLA otonom olarak (Docker API üzerinden) yeniden başlatamaz veya güncelleyemez.
3. **İş Mantığı Sınırı:** Çağrı yönlendirme (Dialplan), SAGA veya AI (LLM) iş mantıklarına KARIŞAMAZ. Sadece Docker konteynerlerinin CPU/RAM/GPU sağlığını izler, gerekirse karantinaya alır ve `UPSTREAM_URL`'e rapor basar.
4. **Loglama:** Standart `println!` yasaktır. SUTS v4.0 JSON (Tracing) zorunludur.
